What Is This System ?


  • Managing and controlling the projects of students across all departments using manual or traditional process is a very tedious job. The main aim of this project is to create an automated system for managing all the activities of projects. Project management system is a system for managing, controlling, monitoring the projects of students. It is a web based portal or application which is useful for students, project guide.
  • This syetsm contains 3 users and 3 dashboards
  • the students and staffs are already registered by the admin into the system
  • Apart from this admin can also register the staffs and can also add the domains. Also admin can reply to students in student feedback section.
  • Admin will give students and staff a specific email address and password through which they will be able to access their respective dashboard.
  • Then registered students can login into the system using their id and password to get authenticated.
  • When the students login to the system, then they will see their part of UI i.e. their respective profiles.
  • Students can add their respective iee reports of their projects by chossing the report in pdf format and the guide or staff under whom the project is being perfomed.
  • Also student can manages their added projects can edit them delete them can can view the reports. Also student can search for the project by using domain name or project name
  • student can view the marks assigned to them by their chossen staff or guide in view marks section
  • Students can also send the feedback or queries to admin in student feedback section
  • project guide needs to login into the system using their id and password. When the guide login to the system, then they will see their part of UI i.e. their respective profiles.
  • Admin and Project guide co-ordinates with each other. Where in we have provided an upload form for the students where the students have to fill their project details including the project report, now the form is sent to the To the project guide who in turn verifies the project details as well as the report and marks the report and the marks are sent to the respective student.
